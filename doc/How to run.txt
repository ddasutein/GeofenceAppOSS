## Install the latest version of Android Studio

Windows, macOS, and Linux: https://developer.android.com/studio/

## How to build

1. After installing Android Studio, download the Android 6.0 OR Android 7.0 SDK.

2. Import the project in Android Studio

3. Build project

If you have your own device, you do not need to download an emulator. If you don't have an Android device, download either an Android 6.0 or 7.0 emulator with the Google APIs installed.
